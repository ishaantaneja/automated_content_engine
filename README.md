# Automated Content Engine
# 🧠 Automated Content Engine (Shorts + Reels Generator)

⚡ Fully automated YouTube Shorts & Instagram Reels generator built with Python, Bark, and AI.  
💸 Designed to work while you sleep. Monetize content on autopilot.

---

## 💡 What It Does

> This engine:
- 💬 Generates AI-powered scripts across multiple niches
- 🎙️ Converts scripts to voiceovers using Bark (GPU-accelerated TTS)
- 🎧 Splits + merges audio to bypass Bark’s 13s limitation
- 📁 Outputs clean `.wav` files ready for video sync

---

## 🎯 Niches Supported (So Far)

- 🧠 Motivation
- 🛠️ AI tools
- 📜 History & Crime
- 🔥 Facts & Quotes
- ☀️ Sanatan Dharma / Eastern Wisdom
- 🕉️ Bhagavad Gita / Paramhansa Yogananda

| Tech           | Use                              |

| PyTorch (CUDA) | GPU-accelerated inference        |
| SciPy          | `.wav` file merging              |
